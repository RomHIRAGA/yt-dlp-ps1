この Powershell のスクリプトは、 yt-dlp.exe をGUIで使用することと、一部、年齢制限がある動画に対応したものになっています。
<img width="428" height="200" alt="image" src="https://github.com/user-attachments/assets/b7e176f8-a825-446f-b0bd-ed8967c76de2" />
スクリプトを「yt-dlp.exe」等と一緒のフォルダに入れます
（cookies.txtは取り敢えず無くてもいいです）

「Shiftキーを押しながら右クリック」して、「PowerShellで 実行」 を選んで実行してください。
